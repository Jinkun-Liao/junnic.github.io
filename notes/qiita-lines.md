# 代码行号展示（Qiita风格示例）

> 本笔记演示如何在代码块中显示行号，并提供复制与语言标签。

## Python 示例

```python
import sys

def greet(name: str) -> str:
    return f"Hello, {name}!"

if __name__ == "__main__":
    print(greet("Qiita"))
```

## JavaScript 示例

```javascript
function sum(a, b){
  return a + b;
}
console.log(sum(3, 4));
```
